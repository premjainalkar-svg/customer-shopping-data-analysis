# 📊 End-to-End Retail Customer Analytics Project

## 🚀 Project Overview

This project analyzes retail customer shopping data to generate business insights using Excel, Python, SQL, and Power BI.

The objective of this project is to understand customer purchasing behavior, identify high-value customers, analyze revenue trends, and build an interactive dashboard for decision-making.

---

## 🛠 Tools & Technologies

- **Microsoft Excel** – Initial data cleaning and preprocessing  
- **Python (Pandas, NumPy, Matplotlib, Jupyter Notebook)** – Data transformation & exploratory analysis  
- **SQL Server** – Business queries, aggregations, and customer segmentation  
- **Power BI** – Interactive dashboard & KPI visualization  
## 📌 Project Workflow

### 1️⃣ Data Cleaning
- Removed duplicates and handled missing values
- Standardized text fields
- Created calculated columns
- Prepared structured dataset for analysis

### 2️⃣ Exploratory Data Analysis (Python)
- Revenue distribution analysis
- Customer purchase frequency analysis
- Category performance insights
- Age group segmentation

### 3️⃣ SQL Analysis
- Customer Segmentation (New / Returning / Loyal)
- Monthly Revenue Trend
- Category-wise Revenue
- Total Revenue & Average Billing
- Top Spending Customers

### 4️⃣ Power BI Dashboard
- KPI Cards (Total Revenue, Total Customers, Avg Billing)
- Monthly Sales Trend Chart
- Category Performance Analysis
- Customer Segmentation Visual
- Interactive Slicers & Filters
